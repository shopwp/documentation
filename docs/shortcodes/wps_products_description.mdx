# [wps_products_description]

The description shortcode allows you to display one or more product descriptions in a list or grid. Useful if you only want to show the product description without other components like pricing or buy button.

:::info
For a full list of available shortcode attributes, refer to the [[wps_products]](/shortcodes/wps_products) shortcode.
:::

<p className='live-example-heading'>
	<strong>Live Example:</strong> Show only descriptions for the first six
	products with pagination
</p>

```js
[wps_products_description page_size="6"]
```

<p className='live-example-heading'>
	<strong>Live Example:</strong> Show product pricing by product id
</p>

```js
[wps_products_description product_id="2216876408880"]
```
