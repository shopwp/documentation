# [wps_products_gallery]

The gallery shortcode allows you to display one or more product images in a list or grid. Useful if you only want to show the product images.

:::info
For a full list of available shortcode attributes, refer to the [[wps_products]](/shortcodes/wps_products) shortcode.
:::

<p className='live-example-heading'>
	<strong>Live Example:</strong> Show only images of the first six products
</p>

```js
[wps_products_gallery page_size="6"]
```

<p className='live-example-heading'>
	<strong>Live Example:</strong> Show product images by product id
</p>

```js
[wps_products_gallery product_id="2216876408880"]
```

<p className='live-example-heading'>
	<strong>Live Example:</strong> Show image zoom
</p>

```js
[wps_products_gallery product_id="2216876408880" show_featured_only="true" show_zoom="true"]
```
