# [wps_products_title]

The title shortcode allows you to display one or more product titles in a list or grid. Useful for only show the product title.

:::info
For a full list of available shortcode attributes, refer to the [[wps_products]](/shortcodes/wps_products) shortcode.
:::

<p className='live-example-heading'>
	<strong>Live Example:</strong> Show the first six product titles with
	pagination
</p>

```js
[wps_products_title page_size="6"]
```

<Products
	settings={{
		excludes: ['images', 'pricing', 'description', 'buy-button'],
		query: 'Super*',
		pageSize: 3,
		limit: false,
		pagination: true,
	}}
/>
