# [wps_translator]

The ShopWP translator shortcode comes with the [ShopWP Translator extension](https://wpshop.io/extensions/translator/). It allows you to display a widget for customers to switch between different currencies and languages.

:::info
This shortcode requires the [ShopWP Translator extension](https://wpshop.io/extensions/translator/).
:::

## Shortcode Examples

```js
// Display the translator widget
[wps_translator]

// Fix the widget to the bottom of the page, even during scroll
[wps_reviews fixed="true"]
```

## Available Attributes

| Atrribute | Default | Description                                                                              |
| :-------- | :------ | :--------------------------------------------------------------------------------------- |
| fixed     | `false` | Allows for "fixing" the translator widget to the bottom of the page, even during scroll. |

### fixed

Allows to "fixing" the translator widget to the bottom of the page, even during scroll.

**Example**

```js
[wps_reviews fixed="true"]
```
