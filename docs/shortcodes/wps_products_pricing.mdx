# [wps_products_pricing]

The ShopWP pricing shortcode allows you to display one or more product prices in a list or grid. Useful for only showing product pricing.

:::info
For a full list of available shortcode attributes, refer to the [[wps_products]](/shortcodes/wps_products) shortcode.
:::

<p className='live-example-heading'>
	<strong>Live Example:</strong> Show the first six product prices with
	pagination
</p>

```js
[wps_products_pricing page_size="6"]
```

<Products
	settings={{
		excludes: ['images', 'title', 'description', 'buy-button'],
		query: 'Super*',
		pageSize: 3,
		limit: false,
		pagination: true,
	}}
/>
